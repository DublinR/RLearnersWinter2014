Ordered logistic regression - polr

  • The polr command from the MASS package can be used to estimate an ordered logistic regression model.

  • The command name comes from proportional odds logistic regression, highlighting the proportional odds assumption in our model.

  • polr uses the standard formula interface in R for specifying a regression model with outcome followed by predictors.

  • We also specify Hess=TRUE to have the model return the observed information matrix from optimization (called the Hessian) which is used to get standard errors.

  • Syntax

Model <- polr(ranking ~ variables, data = dat, Hess = TRUE)
